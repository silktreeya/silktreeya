#ISSUE-2026-03

AI 기반 자동 코드 생성 모듈 오류 수정

진행 중 (Open) 우선순위: 높음
📝 이슈 상세 설명
Vibe Coding 모듈에서 특정 프롬프트 입력 시 순환 참조 오류가 발생하고 있습니다. 특히 'Full Stack' 키워드가 포함될 때 백엔드 API 엔드포인트가 중복 생성되는 현상이 발견되었습니다. [재현 방법] 1. 프롬프트 창에 "Build a full stack blog app" 입력 2. 생성된 파일 목록에서 server.js 확인 3. 동일한 포트(3000)를 사용하는 프로세스가 중복 실행됨 [예상 원인] 코드 생성 엔진의 재귀 함수 호출 시 종료 조건이 불분명하여, 동일한 명세서(Specification)를 기반으로 코드를 무한 생성하는 것으로 추측됩니다.
